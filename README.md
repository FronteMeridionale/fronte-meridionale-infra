# fronte-meridionale-infra
Infrastruttura Telegram-first per la partecipazione al Fronte Meridionale.  Il sistema consente agli utenti di entrare tramite Telegram, registrare la propria partecipazione con una transazione verificabile e ottenere uno status (sostenitore o elettore).  Il registro membri è la base per il futuro sistema di voto della Front DAO.
