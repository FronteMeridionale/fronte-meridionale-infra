# fronte-meridionale-infra
Infrastruttura Telegram-first per il registro membri del Fronte Meridionale. Il sistema consente agli utenti di entrare tramite Telegram, registrare la propria partecipazione con una transazione verificabile e ottenere uno status nel registro (sostenitore o elettore). Lo store in-memory è un placeholder: in produzione verrà sostituito da un database persistente.
